# 谁是价格王

一个面向中文用户的猜价格游戏原型，灵感来自 `guesstheprice.net`，但商品、文案和体验都更贴近国内语境。

## 当前进度

目前仓库已经有一个可直接运行的 MVP：

- 中文商品池与价格竞猜逻辑
- 分数、连击、局数统计
- 今日挑战重开
- 适配桌面端和移动端的单页界面
- 纯 Node 静态服务，不依赖脚手架

## 运行方式

直接运行：

```bash
node backend/index.js
```

然后打开 `http://localhost:3000`。

## 项目结构

```text
backend/index.js   # 本地静态服务
index.html         # 游戏页面
styles.css         # 视觉样式
script.js          # 游戏逻辑
frontend/          # 预留给后续 React 化重构
data/              # 预留给后续商品数据维护
docs/              # 预留给后续产品文档
```

## 下一步方向

1. 把商品数据拆到独立 JSON，便于批量维护。
2. 增加“更高/更低”连续闯关模式，更接近上头感。
3. 接入排行榜、每日题库和分享卡片。
4. 再把前端升级为 React/Next.js 工程化版本。
